To run code locally in vscode:

1. Install the jupyter extension
2. Select a kernel in notebook (top right corner of notebook)
3. Run all cells

If you want to run the code from google collab then create a folder called
mlcoursework2_k21008618 in your drive and copy the folders in this repo into
the folder.

Running the code will load up plots that show results of the saved experiments. All models have been trained and results have been saved to the results directory.

To run experiments from the beginning, clear the results directory. To train the resnet18 model from the beginning, clear the simclr directory.

Project Submission:

- plots: stores all the plots generated by the code
- results: stores all the results from experiments
- simclr: stores resnet18 model weights and embeddings generated by model
- ml_coursework_k21008618.ipynb: the code for the coursework. It contains:
  - Algorithm Implementation
  - Results Replication
  - Algorithm Modification
